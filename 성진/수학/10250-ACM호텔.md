## 10250 ACM 호텔


### 문제링크
https://www.acmicpc.net/problem/10250

### 문제풀이 및 주석

#1.단순 연산을 이용한 풀이
```C++
#include <iostream>

using namespace std;

int main() 
{
    int repeat;
    cin >> repeat;
    for (int i = 0; i < repeat; i++)
    {
        int H, W, N;
        cin >> H >> W >> N;
/*
삼항 연산자를 이용하였다! N이H의 배수일 경우에는H의 값이 층이 된다!
마찬가지로 N이H의 배수일 경우 방 번호는 N/H, 그렇지 않으면 +1이다!
*/
        int floor = (N%H==0)?H:N%H;     
        int room = (N%H==0)?N/H:N/H+1;  

        if (room<10)
            printf("%d0%d\n", floor,room);
        else
            printf("%d%d\n", floor,room);
    }
}
```

#3.피드백 후 최종
```C++
#include <iostream>

using namespace std;

int main()
{
    int repeat;
    cin >> repeat;
    for (int i = 0; i < repeat; i++)
    {
        int H, W, N;
        cin >> H >> W >> N;

        printf("%d%02d\n", (N - 1) % H + 1, (N - 1) / H + 1);
    }
}
```
